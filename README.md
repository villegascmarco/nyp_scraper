
# NYP Scraper

Scraping New York Post web page with Python.


## Run Locally

Clone the project

```bash
  git clone https://github.com/villegascmarco/nyp_scraper.git
```

Go to the project directory

```bash
  cd nyp_scraper
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the python file

```bash
  py scrapper.py
```
